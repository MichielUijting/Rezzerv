- Syntax en bundling van aangepaste Admin-regressierunner succesvol.

Restrisico:
- De nieuwe regressies zijn in code toegevoegd; volledige end-to-end uitvoering binnen Admin zelf is afhankelijk van de runtime-omgeving waarin de browserregressietest wordt gestart.
