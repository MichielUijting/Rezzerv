Uitgevoerde checks
1. Frontend-build uitgevoerd met npm run build -> geslaagd.
2. Starttegel toont nu Aankopen.
3. Route /aankopen toegevoegd; /kassabon redirect naar /aankopen.
4. Hoofdschermtitel aangepast naar Aankopen.
5. Kaarten zichtbaar als:
   - Bronnen
   - Nieuwe aankopen
   - Open en verwerk
6. Open werkblad-route behouden via bestaande batchdetailroute.
7. Terugknop vanuit werkblad wijst nu naar /aankopen.

- Bestaande admin/regressiescripts zijn in deze stap niet volledig herijkt op de nieuwe terminologie.
