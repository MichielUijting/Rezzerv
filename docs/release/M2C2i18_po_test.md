# M2C2i-18 PO-test

## Testdoel

Controleren dat meer Lidl-bonartikelen in `Externe databases` een kandidaat/code tonen.

## Teststappen

1. Open de app.
2. Ga naar `Externe databases`.
3. Filter winkelketen op `lid`.
4. Loop pagina 1 t/m 5 door.
5. Controleer vooral eerder ontbrekende kandidaten:
   - Kidneybonen
   - Kipdijfilet blokjes
   - Komkommer
   - Lasagnebladen 500g
   - Peterselie bos
   - Tomatenblokjes
   - Tortilla chips
   - Volkoren pasta
   - Winterpeen

## Verwacht

- De genoemde regels tonen een artikelnummer/code.
- Er wordt niets automatisch toegevoegd aan voorraad of Mijn artikel.
