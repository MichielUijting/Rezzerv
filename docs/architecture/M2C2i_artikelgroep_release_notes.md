# M2C2i Artikelgroep - Release notes

## Samenvatting

Deze branch vervangt de zichtbare frontendterm `Mijn artikel` door `Artikelgroep` zonder database- of parserwijziging.

## Reden

`Mijn artikel` suggereerde een functioneel hoofdbegrip. De nieuwe lijn is dat Rezzerv een artikelnaam toekent en het huishouden optioneel met eigen artikelgroepen kan ordenen.

## Impact

- Uitpakken wordt terminologisch eenvoudiger.
- Productclassificatie blijft bij Externe databases/productcatalogus.
- Voorraadidentiteit blijft technisch.
