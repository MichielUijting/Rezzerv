# M2C2i Artikelgroep - Backend scope

Deze stap wijzigt geen backendlogica.

Backendvelden zoals `matched_household_article_id` blijven technische koppelingen. Zij worden niet hernoemd in deze release, omdat dat een datamodel- en API-contractwijziging zou zijn.

De zichtbare gebruikersbegrippen worden eerst in de frontendlaag aangepast.
