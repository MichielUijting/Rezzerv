# M2C2i Artikelgroep - Vervolg voor Instellingen

De echte instellingenfunctionaliteit valt buiten deze eerste terminologiestap.

## Latere release

Instellingen > Artikelgroepen kan later toevoegen:

- Eigen artikelgroepen gebruiken.
- Artikelgroep tonen bij Uitpakken.
- Artikelgroep tonen bij Voorraad.
- Nieuwe groep toestaan tijdens Uitpakken.
- Standaardgroep per artikel onthouden.

## Guardrail

Artikelgroep blijft huishoudelijke ordening en geen productidentiteit.
