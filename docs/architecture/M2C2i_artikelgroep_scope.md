# M2C2i Artikelgroep - Scope

Deze release is een terminologiestap.

## Wel

- Zichtbare frontendterminologie `Mijn artikel` vervangen door `Artikelgroep`.
- Functioneel vastleggen dat Artikelgroep huishoudelijke ordening is.
- Regressieborging toevoegen voor oude terminologie in de frontendbundle.

## Niet

- Geen databasewijziging.
- Geen artikelgroepentabel.
- Geen instellingenpagina voor Artikelgroepen.
- Geen voorraadmutatie.
- Geen OCR-, parser- of receipt-ingestionwijziging.
- Geen wijziging aan externe kandidaatselectie.

## Vervolg

Een latere release kan Instellingen > Artikelgroepen toevoegen en Artikelgroep als optionele kolom/filter in Uitpakken en Voorraad gebruiken.
