# M2C2i Artikelgroep - PO-acceptatie

De PO-acceptatie voor deze terminologiestap is beperkt:

- Uitpakken toont geen `Mijn artikel`.
- De gebruiker ziet `Artikelgroep` waar eerder `Mijn artikel` stond.
- Koppelen, locatie kiezen en verwerken blijven functioneel gelijk.
- Er zijn geen nieuwe instellingen nodig in deze stap.
- Er wordt niets gemerged zonder expliciete PO-go.
